Dashboard interativo desenvolvido para centralizar e visualizar minha trajetória de aprendizado contínuo. O sistema organiza certificações por categoria, instituição e cronologia, utilizando gráficos para análise estatística do meu desenvolvimento profissional.

## 🚀 Tecnologias e Arquitetura
Este projeto foi concebido com foco em **performance e simplicidade**, eliminando processos complexos de build para garantir uma entrega rápida e leve.

* **Core:** Vue 3 (Composition API) via CDN.
* **Visualização de Dados:** Chart.js para dashboards de categorias e instituições.
* **Reatividade:** Filtros dinâmicos baseados no *ledger* de certificados.
* **Deploy:** Integração direta com GitHub Pages.

## 📈 Por que este projeto?
Como estudante de Sistemas de Informação, acredito que a transparência sobre a evolução técnica é fundamental. Este dashboard não apenas lista conquistas, mas permite uma visão analítica do meu perfil profissional (com foco em ESG, Dados e Tecnologia).

## 🛠 Como utilizar
1. Acesse o [link oficial](https://gabrielteramae.github.io/certificados-listados/).
2. Utilize os filtros na interface para explorar os certificados por áreas de interesse.
